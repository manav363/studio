# **App Name**: SkillBridge

## Core Features:

- User Authentication and Profiles: Secure user registration, login, and comprehensive profile creation with skills, portfolios, and experience.
- Skill-Based Matching: Enable clients to find freelancers using skill-based filters, price ranges, and ratings.
- Direct Messaging: Facilitate direct communication between clients and freelancers via in-platform messaging with file sharing capabilities.
- Custom Proposal System: Freelancers create service listings and clients can solicit custom proposals.
- Review and Rating System: Allow clients to rate freelancers after project completion, contributing to trust and discovery.
- AI-Powered Skill Suggester: Suggest relevant skills for freelancers to add to their profiles using an AI tool.
- Admin Moderation: Enable administrators to view and remove users or listings, and review reported content.

## Style Guidelines:

- Primary color: Slate Blue (#708090) for a professional and trustworthy feel.
- Background color: Light Gray (#D3D3D3) to ensure readability and a clean interface.
- Accent color: Sandy Brown (#F4A460) to highlight important elements and calls to action.
- Body and headline font: 'Inter' (sans-serif) for a modern and neutral look, ensuring readability across the platform.
- Use clear, minimalist icons to represent skills and categories.
- Maintain a clean and organized layout with clear hierarchy to enhance usability.
- Subtle transitions and loading animations to enhance user experience without being distracting.
